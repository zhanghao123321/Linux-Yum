# Linux-Yum源
### 一键安装脚本
```shell
# 国内使用
bash <(curl -sSL https://raw.githubusercontent.com/zhanghao123321/Linux-Yum/main/ChangeMirrors.sh)
# 教育网用
bash <(curl -sSL https://raw.githubusercontent.com/zhanghao123321/Linux-Yum/main/ChangeMirrors.sh) --edu
# 海外使用
bash <(curl -sSL https://raw.githubusercontent.com/zhanghao123321/Linux-Yum/main/ChangeMirrors.sh) --abroad
```
